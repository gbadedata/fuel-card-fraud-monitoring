# Data

There is no public labelled fuel-card fraud dataset; issuers do not release one. This
project runs on a schema-faithful mock generated by `fuelguard.fuel_data.mock_fuel_frame`,
which models fleets running realistic corridors and injects the fraud typologies a
fuel-card risk team sees.

To run on your own feed, drop a CSV at `data/transactions.csv` with these columns:

    ts, card_id, driver_id, fleet_id, hub, state, lat, lon, product, gallons,
    unit_price, amount, odometer, entry_mode, tank_capacity, is_fraud

`product` is one of diesel, unleaded, premium, merchandise, cash. `is_fraud` is optional
for scoring and required only for evaluation. Raw CSVs are git-ignored.
